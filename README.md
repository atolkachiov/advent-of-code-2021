# C++ solutions of AoC 2021 problems
https://adventofcode.com/2021

## Requirements to run 
* g++ 7.5.0+ 
* boost or install it: 
```
sudo apt-get install libboost-all-dev
```

## Run
From the repo root: `./run.sh <task_number>` where task number matches c++ file name.
```
./run.sh 02-07
./run.sh 17
```
etc.
